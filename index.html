Plugin based architecture

Nemo uses a plugin based architecture to allow for a lightweight core module, customization, and the opportunity for you to contribute easily to the Nemo ecosystem.

Clever Configurability

Using the clever, concise and powerful confit configuration module and its shortstop handler syntax, you have many elegant ways to configure and override.

Easy to intergrate

Nemo is an unopinionated framework and can be used with mocha, jasmine, cucumberjs or any other framework. Nemo can also be used as a stand-alone tool for convenience scripts, or paired with test runner/task runner modules to create a UI test automation suite.
